# 彩票号码预测系统 - 规则匹配策略

## 核心发现：lag1_zone=0 规则

### 规则定义
当上期（第N-1期）第7号在1-10区间时，预测下期（第N期）第7号

### 回测结果
- 验证命中率：22.2% (4/18)
- 随机基准：10.2%
- 超额：+12.0%

### 命中详情
| 期数 | 上期第7号 | 实际开出 | 是否命中 |
|------|----------|--------|---------|
| 29期 | 3 | 1号 | OK |
| 46期 | 1 | 3号 | OK |
| 81期 | 3 | 17号 | OK |
| 104期 | 6 | 1号 | OK |

### 105期预测（基于104期结果）
- 104期第7号 = 1号
- 上期（103期）第7号 = 6号（1-10区间）
- 规则匹配：lag1_zone=0
- 排除104期开出的1号

### 核心逻辑
